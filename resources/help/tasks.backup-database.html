Datenbank sichern

Dieser Task erstellt ein vollständiges SQL-Backup der Datenbank und bereinigt automatisch ältere Backup-Dateien.

Command

php artisan db:backup

Ablauf

  1. mysqldump auf dem System suchen via where (Ermittlung Windows/Linux)
  2. Dump der konfigurierten Datenbank erstellen
  3. Backup-Datei unter storage/app/backup/database/ speichern
  4. Backup-Dateien älter als 14 Tage löschen

Backup-Datei

Die Dateien werden nach folgendem Schema benannt: 

storage/app/backup/database/{datenbankname}_{YYYYMMDD_HHmmss}.sql

Bereinigung

Nach dem erfolgreichen Backup werden Dateien im Verzeichnis backup/database gelöscht, deren Änderungsdatum älter als 14 Tage ist. Gelöschte Dateien werden im Log festgehalten.

Logging und Monitoring

  • Bei erfolgreichem Backup wird Dateiname, Datenbankname, Dateigrösse und Dauer geloggt.
  • Gelöschte alte Backups werden separat mit Anzahl und Dateinamen geloggt.
  • Bei einem Fehler werden Fehlermeldung, Exit-Code und Dauer geloggt.

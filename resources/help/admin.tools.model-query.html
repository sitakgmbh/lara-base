Model Query

Das Model Query Tool ermöglicht das gezielte Abfragen und Filtern von Datenbankeinträgen aus verschiedenen Datenmodellen. Es richtet sich an Administratoren, die ohne direkten Datenbankzugriff schnell Daten einsehen, prüfen oder exportieren möchten.

Model auswählen

Im oberen Bereich kann aus einer Auswahlliste ein Datenmodell gewählt werden. Zur Verfügung stehen ausschliesslich freigegebene Modelle.

Nach der Auswahl werden die verfügbaren Spalten des Modells automatisch geladen und stehen für Filterung und Sortierung zur Verfügung.

Sortierung

Über die Felder Sortierung und Richtung kann die Ergebnisliste nach einer beliebigen Spalte auf- oder absteigend sortiert werden.

Filter

Mit dem Button Filter können beliebig viele Filterbedingungen hinzugefügt werden. Jeder Filter besteht aus drei Teilen:

  • Spalte – die zu filternde Datenbankspalte
  • Operator – die Art der Bedingung
  • Wert – der Vergleichswert (entfällt bei IS NULL / IS NOT NULL)

Folgende Operatoren stehen zur Verfügung:

  • = – exakte Übereinstimmung
  • != – ungleich
  • LIKE – Teilstring-Suche (z. B. muster findet mustermann)
  • > / < – grösser / kleiner als
  • IS NULL – Feld ist leer
  • IS NOT NULL – Feld ist nicht leer

Mehrere Filter werden als UND-Bedingung kombiniert. Einzelne Filter können jederzeit mit dem roten X-Button entfernt werden. Mit Reset werden alle Filter, Sortierungen und die Seitenposition zurückgesetzt.

Ergebnisse

Die Treffer werden unterhalb der Filter in einer Tabelle dargestellt. Alle Spalten des gewählten Modells werden angezeigt. Über das Auswahlfeld Einträge kann die Anzahl der pro Seite angezeigten Datensätze angepasst werden (10, 25, 50, 100 oder alle).

Export

Mit dem Download-Button wird die aktuelle Abfrage — inklusive aller aktiven Filter und Sortierungen — als CSV-Datei exportiert. Der Export enthält alle Treffer, unabhängig von der gewählten Seitenanzahl. Einige Felder wie z. B. restore_context oder profile_photo_base64 werden aus dem Export ausgeschlossen, da sie technische oder sehr grosse Inhalte enthalten:

Die Datei wird im CSV-Format mit Semikolon als Trennzeichen und UTF-8 exportiert.

Aufgabenplanung

Die Aufgabenplanung dient zur Einsicht und Steuerung der im System hinterlegten geplanten Hintergrundprozesse. Einige Prozesse werden in regelmässigen Intervallen automatisch ausgeführt (z. B. Datensynchronisationen, Prüfungen, Backups).

Übersicht

Alle konfigurierten Aufgaben werden in einer Tabelle angezeigt. Zu jedem Eintrag werden folgende Informationen dargestellt:

  • Task – Name des Artisan Commands
  • Beschreibung – Kurzbeschreibung des Befehls
  • Intervall – menschlich lesbarer Ausführungsrhythmus (z. B. „Täglich um 02:00 Uhr")
  • Nächster Lauf – geplanter Zeitpunkt der kommenden Ausführung

Die Tabelle kann durch Klick auf die Spaltenköpfe sortiert werden. Über den Pfeil am Zeilenanfang lässt sich eine Detailansicht mit vollständiger Beschreibung und Intervall aufklappen.

Manuell ausführen

Jede Aufgabe kann über den Button am Zeilenende manuell gestartet werden.

Ausgabeanzeige

Nach dem Start erscheint automatisch ein Dialogfenster mit folgenden Informationen:

  • gestarteter Befehl
  • Startzeitpunkt
  • Dauer der Ausführung
  • Konsolenausgabe des Commands

Fehler werden ebenfalls direkt im Ausgabedialog angezeigt, einschliesslich der originalen Fehlermeldung.
